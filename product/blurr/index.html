freescale i.MX6简介

  • 四核ARM Cortex A9
  • 64bitDDR3内存
  • 3D图形加速
  • 1080P高清视频编解码
  • HDMI、LVDS、LCD多种屏幕接口，显示屏支持同时1~4输出；摄像头输入
  • 千兆以太网、USB、PCle、SATA
  • CAN x 2/UART x 5/SD x 4/SPI x 4/12C x 3/IIS x2

核心板特色

  • 集成CPU、4xDDR3、Flash、电源管理
  • 但电源供电，可用锂电池供电
  • 兼容单核、双核、四核，为产品设计带来极大灵活性
  • 保留CPU但全部IO
  • 面积小68*52mm,可用于移动设备
  • 提供常用外设扩展的原理图，便于开发

芯片特色

  • 最高四核1.2G，汽车级温度
  • 四核+DDR3，性能在
  • BGA624引脚，接口齐全，支持绝大多数应用场景
  • IO兼容1.8/3.3v
  • 高度集成，芯片内涵PHY(除以太网)，系统成本较低
