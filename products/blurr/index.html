Blurr核心板

基于freescale i.MX6

同一平台兼容单核、双核、四核，
业界最齐全的外设，为您的应用平台带来无限的想象空间

freescale i.MX6芯片特色

  • 兼容单核、双核、四核 ARM Cortex A9，支持汽车级

  • 支持同时1~4路输出,每路可显示不同内容

  • BGA624引脚，接口齐全，支持绝大多数应用场景

  • 高度集成，芯片内涵PHY(除以太网)，系统成本较低

freescale i.MX6简介

  • 3D图形加速

  • 1080P高清视频编解码

  • HDMI、LVDS、LCD多种屏幕接口，显示屏支持同时1~4输出；摄像头输入

  • 千兆以太网、USB、PCle、SATA

  • CAN x 2/UART x 5/SD x 4/SPI x 4/12C x 3/IIS x2

核心板特色

  • 集成CPU、4xDDR3、Flash、电源管理

  • 单电源供电，可用锂电池供电

  • 兼容单核、双核、四核，为产品设计带来极大灵活性

  • 保留CPU全部IO

  • 面积小68*52mm,可用于移动设备
